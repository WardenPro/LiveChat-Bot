### US-010: Backfill tests for message command modules
Status: in_progress
Depends on: US-001, US-002, US-006

Description:
As a maintainer, I want unit tests for message command modules and worker logic so that overlay message flow remains reliable.

Acceptance Criteria:

- [ ] Validate payload construction, scheduling/stop interactions, and translation handling through isolated tests.

- [ ] Negative case: empty or blocked message input follows refusal/error path with no outbound side effect.
- [ ] Message modules are fully represented in pnpm test:unit:matrix output.
