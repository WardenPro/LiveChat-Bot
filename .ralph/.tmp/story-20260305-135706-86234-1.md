### US-001: Set up deterministic unit-test foundation
Status: in_progress
Depends on: None

Description:
As a maintainer, I want a stable unit-test toolchain and scripts so that module-level tests can run consistently on local and CI environments.

Acceptance Criteria:
- [ ] Install required dev dependencies with explicit commands: pnpm add -D vitest @vitest/coverage-v8.
- [ ] Add package scripts test:unit and test:unit:coverage, plus initial Vitest config tuned for a Node TypeScript backend.
- [ ] Example: running pnpm test:unit with a sample module test returns exit code 0 and prints at least one passing test.
- [ ] Negative case: a deliberately failing assertion in a sample test causes pnpm test:unit to exit non-zero.
- [ ] pnpm lint and pnpm build still pass after test framework setup.
